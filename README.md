# AliOS

AliOS is a local-first personal life management system.

## Stage 2

Application Shell is implemented:

- Vite + React + TypeScript
- Tailwind CSS
- shadcn/ui-compatible components
- Vazirmatn font
- React Router with hash routing
- AppShell
- Responsive collapsible Sidebar
- Mobile Drawer navigation
- Topbar placeholders
- Placeholder routes:
  - `/`
  - `/today`
  - `/projects`
  - `/journal`
  - `/knowledge`
  - `/settings`

## Development
