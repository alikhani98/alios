# AliOS Roadmap

## Version 1.0

- Local-first app
- Home
- Today
- Projects
- Journal
- Knowledge
- Settings
- CRUD
- Simple search
- Manual backup / restore
- No AI integration

## Version 1.1

- Decision Engine
- Personal Manual
- Master Prompt
- Weekly Review
- Better dashboard

## Version 1.2

- Manual AI workflow
- Generate prompt
- Copy to ChatGPT manually
- Paste response
- Save response to Knowledge

## Version 2.0

- Optional local AI via Ollama adapter
- AIProvider activation
- Journal summary
- Knowledge assistant
- Decision assistant

## Version 2.5

- Optional Tauri desktop app
- SQLite local storage
- File-system backup

## Version 3.0

- Optional PHP/MySQL sync
- Remote backup
- Optional external AI provider
